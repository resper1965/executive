# Changelog

Todas as mudanças notáveis neste projeto serão documentadas neste arquivo.

O formato é baseado em [Keep a Changelog](https://keepachangelog.com/pt-BR/1.0.0/),
e este projeto adere ao [Versionamento Semântico](https://semver.org/lang/pt-BR/).

## [1.1.0] - 2025-12-26

### Adicionado

- Painel Super Admin (`/platform-admin`) para gestão de tenants
- Dashboard com métricas globais de tenants e receita
- Gestão de tenants: Lista, Criação e Detalhes
- Ações administrativas: Desativar/Ativar acesso de tenants

## [1.0.0] - 2025-12-24

### Adicionado

- Portal profissional completo para Sabrina Barros
- Página de Serviços com 4 pilares estratégicos
- Ferramenta de Diagnóstico de Maturidade interativa
- Checklist RN 623/2024 com progresso visual
- Blog com 34+ artigos sobre saúde suplementar
- Integração com Google Analytics 4
- Logos de parceiros clicáveis (Bekaa, Ionic Health, Baalbeck)
- Design responsivo e premium com Tailwind CSS
- Animações suaves com Framer Motion

### Modificado

- Humanização completa dos textos do blog (remoção de padrões de IA)
- Otimização de imagens e assets
- README profissional com documentação completa

### Removido

- Arquivos legados não utilizados
- Imagens duplicadas do diretório public/
- Pastas obsoletas (logos/, scripts/, spec-kit/)
